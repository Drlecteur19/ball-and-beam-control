# Ball and Beam Control System

This project simulates and controls a Ball and Beam system using Python.

## Objectives
- Model the system dynamics
- Simulate open-loop behavior
- Implement PID control
- Build a physical prototype
- Explore advanced control methods

## Open Loop Model

The governing equation is:

r'' = -(5/7)g sin(theta)

where:
- r = ball position
- theta = beam angle
